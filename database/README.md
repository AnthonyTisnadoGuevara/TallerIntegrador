# Base de datos - Gestión de Sílabos

Esta carpeta contiene los scripts SQL para crear la base de datos del MVP del sistema de monitoreo de mejora continua.

## Tablas principales

- `silabos`: registra los datos generales de cada sílabo.
- `validacion_silabo`: registra si el sílabo cumple con las secciones obligatorias.
- `historial_silabo`: guarda los cambios de estado y observaciones.
- `alertas_silabo`: almacena alertas generadas para sílabos pendientes, incompletos u observados.

## Secciones obligatorias del sílabo

1. Datos generales
2. Sumilla
3. Aporte al perfil de egreso
4. Programación por unidades de aprendizaje
5. Estrategias metodológicas
6. Recursos y escenarios de enseñanza-aprendizaje
7. Técnicas e instrumentos de evaluación
8. Estrategias de tutoría y apoyo pedagógico
9. Bibliografía
